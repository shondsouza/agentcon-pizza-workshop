# Acerca de Este Taller  

Este taller es parte de la iniciativa **[Global AI Community](https://www.globalai.community)**.  
Nuestra misión es conectar a entusiastas de la IA, desarrolladores y profesionales de todo el mundo a través de recursos de aprendizaje accesibles, talleres prácticos y eventos comunitarios.  

El **Taller de Microsoft Foundry** fue creado para ayudar a los participantes a:  
- Obtener experiencia práctica con Microsoft Foundry  
- Aprender cómo construir agentes inteligentes paso a paso  
- Comprender casos de uso prácticos para prompts del sistema, RAG, llamado de herramientas y MCP  
- Colaborar y compartir conocimientos en un entorno comunitario abierto  

Creemos en **aprender haciendo**—por eso esta sesión está diseñada como un taller práctico guiado donde te irás con un agente de IA funcional que podrás desarrollar.
