# Obtener una Suscripción de Azure

En caso de que no tengas una suscripción de Azure.

## Prueba gratuita de Azure

Puedes obtener una suscripción de $200 si [te registras en Azure](https://azure.microsoft.com/es-es/pricing/purchase-options/azure-account).


## Azure Classroom

::: tip NOTA
Esta oferta solo aplica para eventos proporcionados por la Global AI Community en eventos como AgentCon.
:::

Puedes seguir las instrucciones a continuación y obtener una suscripción de Azure con $10 de créditos válidos por 5 días.

### Paso 1 - Solicitar suscripción
- Navega a: [https://aka.ms/JoinEduLab](https://aka.ms/JoinEduLab)
- Inicia sesión con una cuenta de Microsoft o GitHub

![Azure Classroom Signup Screenshot](/public/azure-classroom/001.png)

- Introduce <!--@include: ./variables/azure-code.md--> el `código de invitación` y `haz clic en canjear`
- Espera a que llegue el siguiente correo electrónico

### Paso 2 - Activar suscripción
- Busca el siguiente correo electrónico en tu bandeja de entrada.
![You have a request to take ownership of an Azure subscription](/public/azure-classroom/002.png)
- Haz clic en el botón `Accept Ownership`
- Inicia sesión con la misma cuenta del paso 1

![](/public/azure-classroom/003.png)
- Después de iniciar sesión, haz clic en `Accept` para iniciar sesión en el tenant de Open Hack.

![](/public/azure-classroom/004.png)
- Puedes usar el nombre de suscripción sugerido o crear el tuyo propio.
- Haz clic en `Review + Accept`

![](/public/azure-classroom/005.png)
- Haz clic en `Accept`
- Tu suscripción se está creando

### Paso 3 - ¡Feliz hacking!
- Puede tomar unos minutos crear tu suscripción.
![](/public/azure-classroom/007.png)

- Cuando tu suscripción esté lista recibirás un correo electrónico con el asunto: **Your Azure subscription is ready**
![](/public/azure-classroom/006.png)

- También puedes encontrar el estado del proceso de creación de la suscripción bajo notificaciones.
- Haz clic en "Go to subscription" en la notificación o `Go to the Azure portal` en el correo electrónico

¡Feliz hacking!

### Consejos y Trucos

#### ¿No puedes encontrar tu suscripción?
- Tu suscripción se crea en el Tenant/Directorio: "Global AI Community"
- A veces necesitas cambiar de tenant antes de ver la suscripción. [Haz clic aquí](https://portal.azure.com/#settings/directory) para cambiar de Directorio.
- A veces necesitas ajustar tu [filtro de Suscripción](https://portal.azure.com/#view/Microsoft_Azure_Billing/SubscriptionsBladeV2) para que aparezca la suscripción.
